# kickstart.nvim - cjb543

Personal Neovim config based on [kickstart.nvim](https://github.com/nvim-lua/kickstart.nvim). Single-file, documented. Adds Discord rich presence via [cord.nvim](https://github.com/vyfor/cord.nvim).

## Plugins

Standard kickstart config plus:

- **cord.nvim** — Discord rich presence

## Requirements

- Neovim (latest stable or nightly)
- `git`, `make`, `unzip`, C compiler (`gcc`)
- [ripgrep](https://github.com/BurntSushi/ripgrep#installation)
- Nerd Font

Then run `nvim`,  Lazy will bootstrap and install everything automatically.
